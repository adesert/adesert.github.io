一个租房分配模型

  • 前置条件：合理认识，合理预估的前提(但不是绝对)

    问题

    两人或三人合租一套公寓，有不同的房间，主卧，次卧等，如何分摊房租的问题
    条件就是：每个人情况不同，在只考虑价格情况下只要自己可接受范围内，主卧，次卧都可以
    两室一厅的公寓，总房租一共550元，两个房间，主卧1,次卧2，租客A,B
    A,B 分别提出一个方案，A1,A2 ; B1,B2
    A1 是 A 提出的住1房间的可接受价格，A2 是A 提出住2房间的可接受价格
    所以：A1+A2 = B1+B2 = 550
    谁对1房间出价高，谁住1房间
    但是房1租金则是：(A1+B1)/2
    房2租金则是：550-(A1+B1)/2

    举例：
    A1=310,A2=240
    B1=290,B2=260
    则 f1 = (A1+B1)/2 = 300 ; f2 = 550-300 = 250
    则结果：A的房租比自己提的数目小10，B 的房租也比自己愿意出的少10，都觉得自己占了便宜
